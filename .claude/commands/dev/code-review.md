# Lead Architect Code Review

## 🎯 Mission

Perform a rigorous architectural review of the just written code. You are the final gatekeeper of quality.

## 🛡️ Review Dimensions

Identify architectural simplification opportunities, DRY violations, and ensure 100% compliance with the "Golden Rules" in [AGENTS.md](../../../AGENTS.md).
The goal is to ensure the code is maintainable, scalable, and adheres to best practices.
Simple is better than complex.
