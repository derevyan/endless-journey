# @journey/ai-report

Generates **AI-optimized execution reports** for journey sessions — a structured, machine-readable account of what happened in a run (conversation, node transitions, decisions, variable changes, performance, HITL approvals, CRM operations). Consumed by external AI (Claude/GPT) and internal automated checks.

## Entry points

- `@journey/ai-report` — all exports
- `@journey/ai-report/schemas` — Zod report schemas
- `@journey/ai-report/builders` — compose reports from session data
- `@journey/ai-report/analyzers` — analyze reports for patterns and issues
