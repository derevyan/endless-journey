# @journey/mindstate

Theory-of-Mind–inspired state tracking: a main agent plus specialized sub-agents (emotion, cognition, motivation) infer each user's emotional, cognitive, and motivational state as a conversation unfolds, so flows can adapt to how someone actually feels.

Documentation lives in `docs/mindstate/README.md`.
